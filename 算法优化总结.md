# 碳达峰预测算法优化总结

## 问题描述
原预测结果显示总能耗突然陡增，不符合实际情况。

## 优化方案
根据`三年的数据.md`和`碳达峰两种模型.md`文档，重新设计算法和参数体系。

---

## 主要修改

### 1. LEAP模型算法重写 (`backend/model/leap_model.py`)

**核心逻辑改进：**
- **能耗计算**：采用简洁公式 `能耗 = GDP × 能源强度(EI)`
- **能源强度**：每年按 `efficiency_improvement_rate` 递减，确保平滑过渡
- **能源结构调整**：
  - 煤炭占比逐年下降（`coal_reduction_rate`）
  - 可再生能源占比逐年上升（`ren_share_increase`）
  - 油气按原比例填充剩余空间
- **排放计算**：
  - 直燃部分：`CO2_dir = E_dir × CF_dir`
  - 电力部分：`CO2_pow = E_pow × EF_grid`
  - 总排放：`CO2 = CO2_dir + CO2_pow`

**关键改进：**
- 移除了复杂的"历史趋势过渡期"逻辑，避免能耗突增
- 使用标准排放因子（tCO2/toe）：煤炭2.5、石油2.1、天然气1.6、可再生0.0
- 可再生能源排放因子为0，真正体现清洁能源的减排作用

### 2. Kaya模型算法重写 (`backend/model/kaya_model_fixed.py`)

**统一参数体系：**
- 采用与LEAP相同的参数逻辑
- 使用 `coal_reduction_rate` 替代 `fossil_reduction_rate`
- 添加 `ren_share_increase` 参数
- 简化计算流程，确保与LEAP一致

**Kaya恒等式：**
```
CO2 = P × (GDP/P) × (E/GDP) × (CO2/E)
    = P × GDP_per_capita × EI × CF
```

### 3. 参数体系调整

**新参数列表：**
- `gdp_growth_rate`：GDP年增长率
- `population_growth_rate`：人口年增长率
- `efficiency_improvement_rate`：能源效率改善率（能源强度年下降率）
- `coal_reduction_rate`：**煤炭**占比年下降率（百分点）
- `ren_share_increase`：**可再生能源**占比年提升率（百分点）
- `power_share_increase`：电力占比年提升率（百分点）

**参数说明：**
- 使用**百分点**（percentage points）而非相对百分比
- 例：`coal_reduction_rate = 0.01` 表示煤炭占比每年下降1个百分点
- 例：`ren_share_increase = 0.015` 表示可再生能源占比每年提升1.5个百分点

### 4. 前端界面更新 (`frontend/src/components/ScenarioManager.vue`)

**参数表单修改：**
- 将"化石能源占比降低率"改为"煤炭占比下降率"
- 新增"可再生能源占比提升率"参数
- 更新参数说明和默认值

**情景列表表格：**
- 显示新的参数列（煤炭下降率、可再生提升率）
- 移除旧的化石能源参数列

### 5. 后端API更新 (`backend/app.py`)

**参数映射：**
```python
scenario_params = {
    'model_type': 'leap',  # 或 'kaya'
    'gdp_growth_rate': 0.05,
    'population_growth_rate': 0.00,
    'efficiency_improvement_rate': 0.03,
    'coal_reduction_rate': 0.01,  # 新参数
    'ren_share_increase': 0.015,  # 新参数
    'power_share_increase': 0.02,
    'grid_cf_decline': 0.05
}
```

### 6. 图表数据结构优化 (`backend/app.py` 和 `frontend/src/components/PredictionResults.vue`)

**能源结构图表调整：**
- **原结构**：煤炭、油气、可再生能源（电力单独）
- **新结构**：煤炭、可再生能源、其他（油气合并）

**图表显示：**
- 堆叠柱状图：显示三类能源的消费量（万吨标煤）
- 折线图：显示三类能源的占比（%）
- 颜色方案：
  - 煤炭：灰色 `#777`
  - 可再生能源：绿色 `#67C23A`
  - 其他（油气）：蓝色 `#409EFF`

## 算法验证要点

1. **能耗连续性**：能源消费应随GDP增长和能效改善平滑变化，不应出现突变
2. **结构约束**：煤炭 + 可再生 + 油气 = 100%
3. **排放逻辑**：可再生能源增加应导致碳排放下降
4. **达峰判定**：满足峰值条件（前高后低，容差0.5%）且后续不再突破
