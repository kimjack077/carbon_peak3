## API使用说明

### 1. 启动后端服务

```bash
cd backend
python app.py
```

默认端口：5000

### 2. API端点

#### 上传数据
```
POST /api/upload
Content-Type: multipart/form-data

file: CSV文件（必须包含total_energy字段）
```

#### 创建情景
```
POST /api/scenarios
Content-Type: application/json

{
  "name": "情景名称",
  "model_type": "kaya",
  "gdp_growth_rate": 0.07,
  "efficiency_improvement_rate": 0.03,
  "fossil_reduction_rate": 0.02,
  "ren_share_increase": 0.02,
  "population_growth_rate": 0.0
}
```

#### 运行预测
```
POST /api/predict
Content-Type: application/json

{
  "scenarios": ["2030年达峰情景", "基准情景"],
  "forecast_years": 36
}
```

#### 获取结果
```
GET /api/results/<scenario_name>
```

### 3. CSV数据格式要求

**必需字段**：
- year: 年份
- gdp: GDP（亿元）
- co2: CO2排放（万吨）
- carbon_intensity: 碳强度
- mix_coal, mix_oil, mix_gas, mix_ren: 能源结构占比（总和=1）
- power_share: 电力占比
- **total_energy: 能源消费总量（万吨标煤）** ← 新增必需

**可选字段**：
- population: 人口（默认1.0）

### 4. 前端调用示例

```javascript
// 1. 上传数据
const formData = new FormData();
formData.append('file', csvFile);

fetch('http://localhost:5000/api/upload', {
  method: 'POST',
  body: formData
}).then(res => res.json());

// 2. 创建情景
fetch('http://localhost:5000/api/scenarios', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    name: '2030年达峰情景',
    model_type: 'kaya',
    gdp_growth_rate: 0.07,
    efficiency_improvement_rate: 0.03,
    fossil_reduction_rate: 0.02,
    ren_share_increase: 0.02
  })
}).then(res => res.json());

// 3. 运行预测
fetch('http://localhost:5000/api/predict', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    scenarios: ['2030年达峰情景'],
    forecast_years: 36
  })
}).then(res => res.json());

// 4. 获取结果
fetch('http://localhost:5000/api/results/2030年达峰情景')
  .then(res => res.json())
  .then(data => {
    // data 是预测结果数组
    console.log(data);
  });
```

## 重要提醒

### 1. 数据格式变化

**新格式要求**：CSV文件必须包含`total_energy`字段

如果使用旧格式数据（没有total_energy），后端会尝试从以下字段获取：
- `total_energy` (优先)
- `energy_consumption` (备用)

### 2. 模型选择

**推荐使用**：`model_type: "kaya"` (修正后的Kaya模型)

**说明**：
- LEAP模型未更新，建议使用Kaya模型
- 所有新情景默认使用Kaya模型

### 3. 参数范围建议

基于真实数据校准：

| 参数 | 保守 | 中等 | 激进 |
|------|------|------|------|
| GDP增长 | 7-8% | 6-7% | 5-6% |
| 能效提升 | 2-3% | 3-4% | 4-5% |
| 化石降低 | 0.5-1% | 2-3% | 5% |

**达峰条件**：`GDP增长 < 能效提升 + 化石下降×0.835`
