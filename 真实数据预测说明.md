# 基于真实三年数据的碳达峰预测算法说明

## 一、数据处理方法

### 1.1 原始数据来源
使用2022-2024年的实际能源消耗和排放数据，包括：
- 综合能源消费量（万吨标准煤）
- 分能源类型消耗：原煤、焦炭、电力、原油、汽油、煤油、柴油、燃料油、天然气、热力、其他能源
- 能源加工转换产出量（万吨标煤）- **负向指标**
- 工业总产值（现价和可比价）
- 二氧化碳排放量合计（万吨CO2当量）

### 1.2 中国标准转换系数

#### 能源标煤当量转换系数（kgce）
根据《中国能源统计年鉴》和国家发改委标准：

| 能源类型 | 转换系数 | 单位 |
|---------|---------|------|
| 原煤 | 1.0 | kgce/kg（已是标煤） |
| 焦炭 | 0.9714 | kgce/kg |
| 电力（终端） | 0.3456 | kgce/kWh |
| 原油 | 1.4286 | kgce/kg |
| 汽油 | 1.4714 | kgce/kg |
| 煤油 | 1.4714 | kgce/kg |
| 柴油 | 1.4571 | kgce/kg |
| 燃料油 | 1.4286 | kgce/kg |
| 天然气 | 1.33 | kgce/m³ |
| 热力 | 0.03412 | kgce/MJ |

#### CO2排放因子（tCO2/tce）
根据IPCC指南和《中国温室气体清单编制指南》：

| 能源类型 | 排放因子 | 说明 |
|---------|---------|------|
| 原煤 | 2.66 | tCO2/tce |
| 焦炭 | 2.86 | tCO2/tce |
| 汽油 | 2.93 | tCO2/tce |
| 煤油 | 2.99 | tCO2/tce |
| 柴油 | 3.06 | tCO2/tce |
| 燃料油 | 3.17 | tCO2/tce |
| 天然气 | 2.16 | tCO2/tce |
| 电力 | 0.5703 | tCO2/MWh（中国电网平均） |
| 热力 | 0.11 | tCO2/GJ（按燃煤供热） |

### 1.3 能源消耗计算

**重要说明**："能源加工转换产出量"是负向指标，表示能源在加工转换过程中的产出部分，需要从总和中扣除。

计算公式：
```
实际综合能耗 = Σ(各类能源标煤当量) - 能源加工转换产出量
```

### 1.4 历史数据趋势分析（2022-2024）

| 指标 | 2022 | 2023 | 2024 | 年均变化 |
|------|------|------|------|---------|
| GDP（可比价，亿元） | 754.26 | 868.07 | 920.14 | +10.54% |
| CO2排放（万吨） | 4818.77 | 5351.33 | 5273.54 | +4.80% |
| 能源消费（万吨标煤） | 980.15 | 1098.94 | 1081.95 | +5.29% |
| 碳强度（万吨CO2/亿元） | 6.39 | 6.16 | 5.73 | -5.27% |

**关键发现**：
1. 经济保持较快增长（年均10.54%）
2. 碳强度持续下降（年均-5.27%），显示经济增长与碳排放逐步脱钩
3. 2023-2024年CO2排放开始下降（-1.45%），显示减排效果开始显现
4. 能源结构：煤炭占83%，可再生能源占17%

## 二、预测模型说明

### 2.1 Kaya恒等式扩展模型

基于Kaya恒等式的扩展模型：
```
CO2 = GDP × (E/GDP) × (CO2/E)
    = GDP × EI × CF
```

其中：
- **GDP**：地区生产总值（亿元）
- **EI**（Energy Intensity）：能源强度（万吨标煤/亿元GDP）
- **CF**（Carbon Factor）：碳因子（万吨CO2/万吨标煤）

### 2.2 碳因子计算

考虑能源结构和电气化：
```
CF = (1 - p) × CF_direct + p × CF_power
```

其中：
- **p**：电力占总能源消费的比例
- **CF_direct**：直燃端碳因子 = Σ(能源占比 × 排放因子)
- **CF_power**：电力端碳因子（考虑电网清洁化）

### 2.3 能源结构分类

将能源分为四大类：
1. **煤炭类**：原煤 + 焦炭
2. **石油类**：原油 + 汽油 + 煤油 + 柴油 + 燃料油
3. **天然气类**：天然气
4. **可再生能源/其他**：其他能源 + 热力 + 部分电力

## 三、预测情景设置

基于历史趋势，设置了四个预测情景：

### 3.1 基准情景（延续当前趋势）

| 参数 | 设定值 | 说明 |
|------|--------|------|
| GDP年增长率 | 8.0% | 略低于历史10.54% |
| 能效提升率 | 3.5%/年 | 略高于历史碳强度下降率 |
| 煤炭占比年降 | 0.5%/年 | 温和去煤化 |
| 可再生能源年增 | 0.8%/年 | 持续增长 |
| 电气化率年增 | 0.2%/年 | 稳步提升 |

**预测结果**：未在2060年前达峰，2060年排放12102万吨

### 3.2 强化政策情景（积极减排）

| 参数 | 设定值 | 说明 |
|------|--------|------|
| GDP年增长率 | 7.5% | 保持较高增长 |
| 能效提升率 | 4.5%/年 | 大力提升能效 |
| 煤炭占比年降 | 1.5%/年 | 大力去煤化 |
| 可再生能源年增 | 1.5%/年 | 快速发展 |
| 电气化率年增 | 0.3%/年 | 加快电气化 |

**预测结果**：
- ✓ 2047年达峰
- 峰值排放：4852万吨（比2024年下降8%）
- 2060年排放：4584万吨（比峰值下降5.5%）

### 3.3 低碳转型情景（加速脱碳）

| 参数 | 设定值 | 说明 |
|------|--------|------|
| GDP年增长率 | 7.0% | 保持中高速增长 |
| 能效提升率 | 5.5%/年 | 最大力度提升 |
| 煤炭占比年降 | 2.5%/年 | 快速去煤化 |
| 可再生能源年增 | 2.2%/年 | 大力发展 |
| 电气化率年增 | 0.4%/年 | 快速电气化 |

**预测结果**：持续下降，2060年排放1159万吨（比2024年下降78%）

### 3.4 延续高排放情景（弱减排）

| 参数 | 设定值 | 说明 |
|------|--------|------|
| GDP年增长率 | 8.5% | 较快增长 |
| 能效提升率 | 2.5%/年 | 较慢提升 |
| 煤炭占比年降 | 0.3%/年 | 缓慢去煤化 |
| 可再生能源年增 | 0.5%/年 | 较慢发展 |
| 电气化率年增 | 0.1%/年 | 缓慢电气化 |

**预测结果**：持续上升，2060年排放22760万吨

## 四、关键发现与建议

### 4.1 关键发现

1. **基准情景无法达峰**：延续当前趋势，排放将持续增长，无法在2060年前达峰。

2. **强化政策可实现2047年达峰**：通过积极的政策措施，可在2047年达峰，峰值比2024年略低。

3. **低碳转型实现持续下降**：采取最积极措施，排放可持续下降，2060年降至1159万吨。

4. **能源结构转型至关重要**：
   - 基准情景：2060年煤炭占比仍有55%
   - 强化政策：2060年煤炭降至29%，可再生达71%
   - 低碳转型：2060年煤炭降至4%，可再生达96%

### 4.2 政策建议

1. **加快能效提升**：
   - 目标：能效年提升率达到4.5-5.5%
   - 措施：技术改造、淘汰落后产能、优化工艺

2. **加速能源结构调整**：
   - 煤炭占比年降1.5-2.5%
   - 可再生能源年增1.5-2.2%
   - 优先发展风电、光伏、水电

3. **推进电气化**：
   - 电气化率年增0.3-0.4%
   - 推广电动交通、电热替代
   - 同步提高电网清洁化水平

4. **强化碳排放管理**：
   - 建立碳排放监测系统
   - 实施碳排放总量和强度双控
   - 建立碳定价机制

## 八、注意事项

1. **能源加工转换产出量**是负向指标，在计算时需要减去，而不是加上。

2. **工业总产值（可比价）**用作GDP指标更适合趋势分析，避免价格因素干扰。

3. **电力排放因子**会随着电网清洁化而下降，模型中已考虑此因素。

4. **能源结构归一化**确保煤炭、石油、天然气、可再生能源占比总和为1。

5. **平滑过渡机制**：预测初期采用历史趋势与目标趋势的加权平均，避免预测值突变。
