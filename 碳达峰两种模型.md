# 通用输入与校验

## 基础数据（用前三年校准）
- 年份：`[Y1,Y2,Y3]`（连续）
- GDP（基础价或不变价，需一致口径）：`[g1,g2,g3]`
- 碳排放总量（tCO₂）：`[c1,c2,c3]`
- 碳强度（tCO₂/GDP）：`[ci1,ci2,ci3]`（若缺可由 `co2/gdp` 反算）
- 能源结构份额（**直燃/最终能**口径，和为 1）：  
  `mix.coal, mix.oil, mix.gas, mix.ren` → 各自三年序列
- 电力需求占比（终端用能中电力占比 0~1）：`power_share` 三年序列

> 建议统一单位：若用 toe 与 kWh 混用，做 `1 toe ≈ 11.63 MWh` 的换算；若企业既有自发电也有外购电，排放分开算再求和。

## 情景参数（对每条情景都给一组）
- `gdp_growth_rate`（年率，例：0.05 = +5%/年）
- `population_growth_rate`（年率；用于 Kaya 里的人口项）
- `efficiency_improvement_rate`（能源效率改善率 = **能源强度**EI 的年**下降**率）
- `coal_reduction_rate`（煤在**直燃份额**中的年下降百分点，例：0.01 = 每年减 1%p）
- `power_share_increase`（电力占比年提升百分点，例：0.02 = 每年 +2%p）
- `ren_share_increase`（可再生份额年提升百分点，优先用于直燃结构，例：0.01）

> 口径说明  
> - “份额提升/下降”用**百分点**（percentage points，p.p.），实现里按 `+/- 0.01` 处理。  
> - 若你希望按**相对增减**（%），把更新式换成乘法（示例里给出两种）。

# 方法 A：LEAP 风格（活动量 × 能效 × 排放因子）

## 逻辑映射到情景参数
- 经济规模：`GDP_{t+1} = GDP_t × (1 + gdp_growth_rate)`
- 能源强度（EI = E/GDP）：`EI_{t+1} = EI_t × (1 - efficiency_improvement_rate)`
- 终端电气化：`power_share_{t+1} = clip(power_share_t + power_share_increase, 0, 1)`
- 直燃燃料结构：
  - 先执行：`coal_{t+1} = max(0, coal_t - coal_reduction_rate)`
  - 再执行：`ren_{t+1}  = min(1, ren_t  + ren_share_increase)`
  - 剩余油、气按“保持相对结构比例”归一：  
    `res = 1 - coal_{t+1} - ren_{t+1}`  
    `oil_{t+1} = res × oil_t/(oil_t+gas_t)`；`gas_{t+1} = res × gas_t/(oil_t+gas_t)`
- 电网排放系数（可选，若要考虑电力低碳化）：可在情景里再加 `grid_cf_decline`，或先固定。
- 排放因子（示意值，需按企业口径替换）：  
  `EF_coal, EF_oil, EF_gas, EF_ren≈0`（tCO₂/单位能耗）  
  `EF_grid`（tCO₂/kWh 或 tCO₂/MWh，注意与能耗口径的一致性）

## 年度计算
1) 基于前三年反推总能耗 `E_t = CO2_t / (Σ mix_f,t × EF_f)`，得到 `EI_t = E_t/GDP_t` 的基期。
2) 预测期逐年：
   - `GDP_t, EI_t → E_t = GDP_t × EI_t`
   - 拆分：`E_pow = E_t × power_share_t`；`E_dir = E_t - E_pow`
   - 直燃排放：`CO2_dir = E_dir × (coal×EF_coal + oil×EF_oil + gas×EF_gas + ren×EF_ren)`
   - 购电排放：若 `E_pow`是**电耗**口径，`CO2_pow = E_pow × EF_grid`；  
     若 `E_pow`是**最终能**口径，需乘以折算系数 `κ`（toe→MWh）。
   - `CO2_t = CO2_dir + CO2_pow`；`CI_t = CO2_t/GDP_t`
3) 达峰判定：找满足 `CO2_{t-1} < CO2_t > CO2_{t+1}` 且后续不再突破的年份（允许±ε容差）。

## 可直接实现的伪代码
```python
def forecast_leap(base, scenario, horizon_year):
    # base: dict 含 years,gdp,co2,mix{},power_share
    # scenario: dict 含 gdp_growth_rate, efficiency_improvement_rate,
    #                     coal_reduction_rate, power_share_increase, ren_share_increase,
    #                     (可选) grid_cf_decline
    # 常量/示意排放因子（替换成企业口径）
    EF = {"coal": 2.5, "oil": 2.1, "gas": 1.6, "ren": 0.0}  # tCO2/（单位能），示意
    EF_grid = 0.65  # tCO2/MWh（示意）；如需逐年下降可加上 scenario["grid_cf_decline"]

    # 1) 基期反推 E 与 EI（用最后一年做起点）
    def E_from_co2_mix(co2, mix):
        denom = sum(mix[f] * EF[f] for f in ["coal","oil","gas","ren"])
        return co2 / max(denom, 1e-9)

    mix_last = {f: base["mix"][f][-1] for f in ["coal","oil","gas","ren"]}
    E_last   = E_from_co2_mix(base["co2"][-1], mix_last)
    EI_last  = E_last / base["gdp"][-1]
    p_last   = base["power_share"][-1]
    g_last   = base["gdp"][-1]

    y = base["years"][-1]
    out = []

    while y < horizon_year:
        y += 1
        # 2) 经济与效率
        g_last *= (1 + scenario["gdp_growth_rate"])
        EI_last *= (1 - scenario["efficiency_improvement_rate"])
        E_t = g_last * EI_last

        # 3) 结构更新
        # 电力占比
        p_last = min(1.0, max(0.0, p_last + scenario["power_share_increase"]))
        # 直燃结构：先煤降、再可再生升、油气按比例补
        coal = max(0.0, mix_last["coal"] - scenario["coal_reduction_rate"])
        ren  = min(1.0, mix_last["ren"]  + scenario["ren_share_increase"])

        remain = max(0.0, 1.0 - coal - ren)
        oil0, gas0 = mix_last["oil"], mix_last["gas"]
        s = max(oil0 + gas0, 1e-9)
        oil = remain * oil0/s
        gas = remain * gas0/s
        mix_last = {"coal": coal, "oil": oil, "gas": gas, "ren": ren}

        # 电网系数可设为常数或按情景下降
        if "grid_cf_decline" in scenario:
            EF_grid *= (1 - scenario["grid_cf_decline"])

        # 4) 排放
        E_pow = E_t * p_last
        E_dir = E_t - E_pow

        CO2_dir = E_dir * (coal*EF["coal"] + oil*EF["oil"] + gas*EF["gas"] + ren*EF["ren"])
        # 若 E_pow 是 MWh，请直接乘 EF_grid；若是 toe，请乘 toe->MWh 的 κ=11.63
        kappa = 1.0  # 按你的口径设定：toe->MWh 就用 11.63
        CO2_pow = (E_pow * kappa) * EF_grid

        CO2_t = CO2_dir + CO2_pow
        CI_t = CO2_t / g_last

        out.append({
            "year": y, "GDP": g_last, "Energy": E_t, "EI": EI_last,
            "power_share": p_last, "mix_coal": coal, "mix_oil": oil, "mix_gas": gas, "mix_ren": ren,
            "CO2": CO2_t, "CI": CI_t, "EF_grid": EF_grid
        })

    # 5) 达峰年
    peak = None
    eps = 0.005  # 0.5% 容差
    for i in range(1, len(out)-1):
        if (out[i]["CO2"] >= (1-eps)*out[i-1]["CO2"]
            and out[i]["CO2"] > out[i-1]["CO2"]
            and out[i]["CO2"] >= (1-eps)*out[i+1]["CO2"]
            and out[i]["CO2"] > out[i+1]["CO2"]):
            peak = out[i]["year"]; break
    return out, peak
```

# 方法 B：Kaya / 扩展 Kaya（顶层分解，显式加入人口项）

标准 Kaya：  
`CO2 = P × (GDP/P) × (E/GDP) × (CO2/E)`  
企业/部门口径也可保留人口项（例如与员工规模/产量相关），你给了 `population_growth_rate`，我们就完整使用：

- 人口（或代表性规模量）：`P_{t+1} = P_t × (1 + population_growth_rate)`
- 人均 GDP：由总 GDP 推得：`GDP_{t+1} = GDP_t × (1 + gdp_growth_rate)`
- 能源强度（EI = E/GDP）：`EI_{t+1} = EI_t × (1 - efficiency_improvement_rate)`
- 综合碳因子（CF = CO2/E）：由**直燃结构** + **电力碳因子**联动情景：
  - 直燃结构按 `coal_reduction_rate`、`ren_share_increase` 更新（与 LEAP 相同）
  - 电力占比按 `power_share_increase` 更新
  - `CF_dir = coal×EF_coal + oil×EF_oil + gas×EF_gas + ren×EF_ren`
  - `CF_el = EF_grid × κ`（与能耗口径一致）
  - **加权综合**：`CF = (1 - power_share) × CF_dir + power_share × CF_el`

> 这样，情景参数直接映射到 Kaya 的两个核心强度项：`EI` 与 `CF`，再与 `GDP`、`P` 合成预测。

## 年度计算
- `CI = EI × CF`  
- `CO2 = GDP × CI`（若希望显式写成人口项，也可 `CO2 = P × (GDP/P) × EI × CF = GDP × EI × CF`，等价）
- 达峰判定同上。

## 可直接实现的伪代码
```python
def forecast_kaya(base, scenario, horizon_year, P0=1.0):
    # P0 可设为基期人口或代表性规模量（不影响 CO2=GDP×EI×CF 的计算，
    # 但便于你输出 P 与人均指标）
    EF = {"coal": 2.5, "oil": 2.1, "gas": 1.6, "ren": 0.0}
    EF_grid = 0.65  # tCO2/MWh（示意）

    # 基期：用最后一年反演 EI 与 CF 起点
    mix_last = {f: base["mix"][f][-1] for f in ["coal","oil","gas","ren"]}

    CI_last = base["co2"][-1] / base["gdp"][-1]  # 若给了 ci[-1] 也可直接用
    CF_dir0 = sum(mix_last[f] * EF[f] for f in mix_last)
    # 电力对 CF 的影响取加权：此处用基期电力占比
    p_last = base["power_share"][-1]
    kappa = 1.0
    CF_last = (1 - p_last)*CF_dir0 + p_last*(EF_grid*kappa)
    EI_last = CI_last / max(CF_last, 1e-9)

    g_last = base["gdp"][-1]
    y = base["years"][-1]
    P = P0

    out = []

    while y < horizon_year:
        y += 1
        # 人口与经济
        P *= (1 + scenario["population_growth_rate"])
        g_last *= (1 + scenario["gdp_growth_rate"])

        # 能源强度与电力、结构
        EI_last *= (1 - scenario["efficiency_improvement_rate"])
        p_last = min(1.0, max(0.0, p_last + scenario["power_share_increase"]))

        # 直燃结构更新
        coal = max(0.0, mix_last["coal"] - scenario["coal_reduction_rate"])
        ren  = min(1.0, mix_last["ren"]  + scenario["ren_share_increase"])
        remain = max(0.0, 1.0 - coal - ren)
        oil0, gas0 = mix_last["oil"], mix_last["gas"]
        s = max(oil0 + gas0, 1e-9)
        oil = remain * oil0/s
        gas = remain * gas0/s
        mix_last = {"coal": coal, "oil": oil, "gas": gas, "ren": ren}

        # 电网系数可按情景下降
        if "grid_cf_decline" in scenario:
            EF_grid *= (1 - scenario["grid_cf_decline"])

        CF_dir = coal*EF["coal"] + oil*EF["oil"] + gas*EF["gas"] + ren*EF["ren"]
        CF = (1 - p_last) * CF_dir + p_last * (EF_grid * kappa)
        CI = EI_last * CF
        CO2 = g_last * CI

        out.append({
            "year": y, "Population": P, "GDP": g_last,
            "EI": EI_last, "CF": CF, "CI": CI, "CO2": CO2,
            "power_share": p_last, "mix_coal": coal, "mix_oil": oil, "mix_gas": gas, "mix_ren": ren,
            "EF_grid": EF_grid
        })

    # 达峰年
    peak = None
    eps = 0.005
    for i in range(1, len(out)-1):
        if (out[i]["CO2"] >= (1-eps)*out[i-1]["CO2"]
            and out[i]["CO2"] > out[i-1]["CO2"]
            and out[i]["CO2"] >= (1-eps)*out[i+1]["CO2"]
            and out[i]["CO2"] > out[i+1]["CO2"]):
            peak = out[i]["year"]; break
    return out, peak
```

## 份额更新的两种口径（任选其一）

- **百分点法（上文默认）**：  
  `s_new = clip(s_old ± Δ[p.p.])`  
  直观、便于政策目标（如“煤每年-1%p，可再生+1.5%p”）。
- **相对比例法（乘法）**：  
  `s_new_raw = s_old × (1 ± r)`；再按所有燃料 `s_new_raw` 归一到 1。  
  更贴近“每年相对变化 r%”。

## 程序接口建议（便于批量情景）

- 输入：`base_data` + `scenarios`（数组），每条情景一组参数；预测到 `horizon`。  
- 输出：每条情景返回**年度表**（year, GDP, EI, CF, CI, CO2, 结构、电力占比、EF_grid）与**达峰年**。  
- 关键校验：  
  - 份额和为 1；不出现负数或 >1；  
  - 单位一致（toe/MWh）；  
  - 若发现 `CO2` 随机尖峰，使用 3 年滑动均值再判峰。

## 怎么用（示例占位）
```python
base = {
  "years":[2022,2023,2024],
  "gdp":[g1,g2,g3],
  "co2":[c1,c2,c3],
  "ci":[c1/g1, c2/g2, c3/g3],  # 或填你已有的
  "mix":{"coal":[...], "oil":[...], "gas":[...], "ren":[...]},
  "power_share":[p1,p2,p3]
}
scn = {
  "gdp_growth_rate": 0.05,
  "population_growth_rate": 0.00,
  "efficiency_improvement_rate": 0.03,
  "coal_reduction_rate": 0.01,
  "power_share_increase": 0.02,
  "ren_share_increase": 0.015,
  # 可选
  "grid_cf_decline": 0.05
}
outA, peakA = forecast_leap(base, scn, 2035)
outB, peakB = forecast_kaya(base, scn, 2035, P0=企业基期人员数或1.0)
```
