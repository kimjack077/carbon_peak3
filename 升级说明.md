# 碳达峰预测系统升级说明

## 升级概述

系统已成功升级为**双模型预测架构**，支持 **LEAP** 和 **Kaya** 两种国际主流碳排放预测模型。

### 主要变更

#### 1. 新增两大预测模型

##### LEAP模型（Low Emissions Analysis Platform）
- **核心公式**：`碳排放 = 活动量 × 能效 × 排放因子`
- **适用场景**：自下而上的详细能源结构分析
- **文件位置**：`backend/model/leap_model.py`

##### Kaya模型（扩展版）
- **核心公式**：`CO2 = 人口 × (GDP/人口) × (能源/GDP) × (CO2/能源)`
- **适用场景**：宏观因子分解和政策效果评估
- **文件位置**：`backend/model/kaya_model.py`

#### 2. 数据格式升级

##### 新格式要求（3年历史数据）
```csv
year,gdp,co2,carbon_intensity,mix_coal,mix_oil,mix_gas,mix_ren,power_share,population
2022,114.37,11200.5,0.0979,0.56,0.18,0.09,0.17,0.27,141175
2023,126.06,11450.8,0.0908,0.54,0.18,0.10,0.18,0.29,140967
2024,134.52,11580.3,0.0861,0.52,0.18,0.11,0.19,0.31,140821
```

##### 字段说明

| 字段名 | 说明 | 单位 | 示例 |
|--------|------|------|------|
| year | 年份 | - | 2024 |
| gdp | GDP总量 | 万亿元 | 134.52 |
| co2 | 碳排放总量 | 万吨CO₂ | 11580.3 |
| carbon_intensity | 碳强度 | tCO₂/万亿元 | 0.0861 |
| mix_coal | 煤炭在直燃能源中占比 | 0-1 | 0.52 |
| mix_oil | 石油在直燃能源中占比 | 0-1 | 0.18 |
| mix_gas | 天然气在直燃能源中占比 | 0-1 | 0.11 |
| mix_ren | 可再生能源在直燃能源中占比 | 0-1 | 0.19 |
| power_share | 电力在终端能耗中占比 | 0-1 | 0.31 |
| population | 人口（可选） | 万人 | 140821 |

**重要约束**：
- `mix_coal + mix_oil + mix_gas + mix_ren = 1`
- 所有占比字段范围：[0, 1]
- 至少需要连续3年的数据

#### 3. 情景参数升级

##### 新增参数

```json
{
  "model_type": "leap",  // 模型类型："leap" 或 "kaya"
  "gdp_growth_rate": 0.05,  // GDP年增长率（5%）
  "population_growth_rate": 0.00,  // 人口年增长率（0%）
  "efficiency_improvement_rate": 0.03,  // 能源效率年改善率（3%）
  "coal_reduction_rate": 0.01,  // 煤炭占比年下降（百分点，1%p）
  "power_share_increase": 0.02,  // 电力占比年提升（百分点，2%p）
  "ren_share_increase": 0.015,  // 可再生能源占比年提升（百分点，1.5%p）
  "grid_cf_decline": 0.05  // 电网碳因子年下降率（5%，可选）
}
```

##### 参数含义详解

**gdp_growth_rate（GDP增长率）**
- 影响：经济规模和总能源需求
- 范围：0 - 0.15（0-15%）
- 典型值：0.05（发达经济体）、0.07（新兴经济体）

**population_growth_rate（人口增长率）**
- 影响：Kaya模型的人口因子
- 范围：-0.02 - 0.05（-2% 到 5%）
- 典型值：0.00（中国当前）、-0.005（人口下降）

**efficiency_improvement_rate（能效改善率）**
- 影响：单位GDP能耗的下降速度
- 范围：0 - 0.10（0-10%）
- 典型值：0.03（积极情景）、0.02（基准情景）

**coal_reduction_rate（煤炭下降率）**
- 影响：煤炭在直燃能源中的占比每年下降的**百分点**
- 范围：0 - 0.10（0-10百分点/年）
- 示例：0.01表示从50%下降到49%（1个百分点）

**power_share_increase（电力占比提升）**
- 影响：电气化水平，电力在终端能耗中的占比每年提升的**百分点**
- 范围：0 - 0.05（0-5百分点/年）
- 示例：0.02表示从30%提升到32%（2个百分点）

**ren_share_increase（可再生能源占比提升）**
- 影响：能源清洁化程度
- 范围：0 - 0.05（0-5百分点/年）
- 示例：0.015表示从20%提升到21.5%（1.5个百分点）

**grid_cf_decline（电网碳因子下降率）**
- 影响：电力生产的碳排放强度
- 范围：0 - 0.10（0-10%）
- 说明：可选参数，不设置则保持恒定

{

## 使用示例

### 1. 准备数据

创建 `base_year_data.csv`：

```csv
year,gdp,co2,carbon_intensity,mix_coal,mix_oil,mix_gas,mix_ren,power_share,population
2022,114.37,11200.5,0.0979,0.56,0.18,0.09,0.17,0.27,141175
2023,126.06,11450.8,0.0908,0.54,0.18,0.10,0.18,0.29,140967

```

### 2. 使用示例数据

```bash
curl -X POST http://127.0.0.1:5000/api/upload/example
```

或通过前端界面点击"使用示例数据"按钮。

### 3. 创建LEAP情景

```python
import requests

scenario = {
    "name": "基准情景-LEAP",
    "model_type": "leap",
    "gdp_growth_rate": 0.05,
    "population_growth_rate": 0.00,
    "efficiency_improvement_rate": 0.03,
    "coal_reduction_rate": 0.01,
    "power_share_increase": 0.02,
    "ren_share_increase": 0.015,
    "grid_cf_decline": 0.05
}

response = requests.post(
    "http://127.0.0.1:5000/api/scenarios",
    json=scenario
)
print(response.json())
```

### 4. 创建Kaya情景

```python
scenario_kaya = {
    "name": "基准情景-Kaya",
    "model_type": "kaya",

    "efficiency_improvement_rate": 0.03,
    "coal_reduction_rate": 0.01,

    "ren_share_increase": 0.015,
}

response = requests.post(
    "http://127.0.0.1:5000/api/scenarios",

```

### 5. 运行预测

```python
predict_request = {
    "scenarios": ["基准情景-LEAP", "基准情景-Kaya"],
    "forecast_years": 30
}

response = requests.post(
    "http://127.0.0.1:5000/api/predict",
    json=predict_request

```

### 6. 获取结果

```python
# 获取图表数据
response = requests.get("http://127.0.0.1:5000/api/chart-data")

# 获取某情景的详细结果

```

## 常见问题

### Q1: 新旧模型预测结果有差异是正常的吗？

**A**: 是的。LEAP和Kaya采用不同的建模方法：
- LEAP更适合详细的能源结构分析
- Kaya更适合宏观政策效果评估
- 两者结果应在合理范围内接近，但不会完全相同

### Q2: 如何选择合适的模型？

**A**: 
- 如果关注**能源结构调整**和**技术路径**，选择LEAP
- 如果关注**驱动因素分解**和**政策贡献度**，选择Kaya
- 建议同时运行两个模型，对比验证

### Q3: 为什么需要3年数据？

**A**: 
- 校准基期参数（能源强度、碳因子等）
- 识别历史趋势
- 提高预测准确性

### Q4: 百分点和百分比的区别？

**A**: 
- **百分点**（percentage point）：绝对差值
  - 从50%到51%是增加1个百分点
- **百分比**（percentage）：相对变化
  - 从50%增长2%是变为51%（50×1.02）

系统参数使用**百分点**更直观。
