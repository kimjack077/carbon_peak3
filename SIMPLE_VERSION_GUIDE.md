# 简化版碳达峰预测系统使用指南

## 概述

简化版碳达峰预测系统是一个轻量级的碳排放预测工具，专注于核心功能：
- 只考虑 **年份、能耗、GDP、碳排放** 四个关键指标
- 提供 **LEAP** 和 **Kaya** 两种预测模型
- 支持自定义场景参数：**GDP增长率**、**能源效率改善率**、**可再生能源占比提升率**

## 特点

### 数据简化
相比完整版，简化版只需要以下数据：
- **year**: 年份
- **energy_consumption**: 综合能源消费量（万吨标准煤）
- **gdp**: 工业总产值（万元）
- **co2_emission**: 二氧化碳排放量（万吨CO₂当量）
- **renewable_ratio**: 可再生能源占比（可选，默认0）

### 模型说明

#### LEAP模型
基于能源强度和碳排放系数：
```
能源消费 = GDP × 能源强度
碳排放 = 非可再生能源消费 × 碳排放系数
```

#### Kaya模型
基于Kaya恒等式：
```
CO2 = GDP × (Energy/GDP) × (CO2/Energy)
```

### 场景参数

1. **模型类型**: 选择LEAP或Kaya模型
2. **生产总值增长率**: 年度GDP增长率（如5% = 0.05）
3. **能源效率改善率**: 能源强度年下降率（如3% = 0.03）
4. **可再生能源占比提升率**: 每年可再生能源占比提升（如1.5% = 0.015）

## 快速开始

### 1. 启动后端服务

```bash
# 在项目根目录
python simple_app.py
```

后端将在 `http://localhost:5001` 启动

### 2. 启动前端服务

```bash
cd frontend
npm run serve
```

前端将在 `http://localhost:8080` 启动

### 3. 使用系统

1. 打开浏览器访问 `http://localhost:8080`
2. 点击 **"简化版预测"** 标签页
3. 在 **"场景设置"** 区域：
   - 输入场景名称（如"积极达峰"）
   - 选择模型类型（LEAP或Kaya）
   - 设置三个参数
   - 点击 **"添加场景"**
4. 重复步骤3创建多个场景（可选）
5. 设置预测年限（默认30年）
6. 点击 **"运行预测"** 按钮

### 4. 查看结果

系统将自动生成三张图表：

#### 碳排放预测曲线
- 显示所有场景的碳排放趋势
- 突出显示碳达峰年份和峰值
- 每个场景用不同颜色区分

#### 生产总值预测
- 显示各场景下GDP增长曲线
- 单位：亿元

#### 能源消费预测（含能源结构）
- 堆叠柱状图显示能源消费总量
- 绿色部分：可再生能源（零排放）
- 灰色部分：非可再生能源
- 显示第一个场景的能源结构变化

## 技术架构

### 后端
- **框架**: Flask
- **端口**: 5001
- **API**:
  - `GET /api/simple/scenarios` - 获取所有场景
  - `POST /api/simple/scenarios` - 创建场景
  - `DELETE /api/simple/scenarios/<index>` - 删除场景
  - `POST /api/simple/predict` - 运行预测
  - `GET /api/simple/historical` - 获取历史数据

### 前端
- **框架**: Vue.js 2
- **图表库**: Chart.js 3
- **端口**: 8080

## 常见问题

### Q1: 为什么碳排放会下降？
A: 当能源效率改善率和可再生能源占比提升速度大于GDP增长带来的能源需求增长时，碳排放会逐渐下降。

### Q2: LEAP和Kaya模型有什么区别？
A: LEAP模型更侧重于能源结构转换，Kaya模型基于恒等式分解碳排放驱动因素。在相同参数下，两个模型可能产生略有不同的结果。

### Q3: 如何调整参数实现更早达峰？
A: 提高能源效率改善率和可再生能源占比提升率，或降低GDP增长率，都能使碳达峰提前。

### Q4: 可以使用自己的数据吗？
A: 可以。修改 `backend/data/simple_data.csv` 文件，按照相同格式填入您的数据即可。
