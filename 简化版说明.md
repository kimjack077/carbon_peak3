# 简化版碳达峰预测系统 - 完整说明

## 📋 系统概述

根据您的需求，我已经创建了一个简化版的碳达峰预测系统，专注于核心功能：

### ✅ 简化内容
- **数据项**：只考虑4个核心指标
  - 年份
  - 综合能源消费量（万吨标煤）
  - 工业总产值（万元）
  - 二氧化碳排放量（万吨CO₂）

- **模型算法**：两种预测模型
  - LEAP模型
  - Kaya模型

- **场景参数**：3个关键参数
  - 生产总值增长率
  - 能源效率改善率
  - 可再生能源占比提升率

### ✅ 核心功能
1. **场景管理**：创建、删除、查看场景
2. **预测计算**：运行LEAP或Kaya模型
3. **结果可视化**：
   - 碳排放曲线图（显示所有场景，标注达峰年份和峰值）
   - 生产总值增长图
   - 能源消费图（堆叠柱状图显示可再生能源和非可再生能源）

## 🎯 使用流程

### 1. 启动系统
**方式一：一键启动**
```bash
双击运行 start_simple.bat
```

**方式二：手动启动**
```bash
# 终端1：启动后端（端口5001）
python simple_app.py

# 终端2：启动前端（端口8080）
cd frontend
npm run serve
```

### 2. 访问系统
打开浏览器访问：http://localhost:8080

点击顶部的 **"简化版预测"** 标签页

### 3. 创建场景
在"场景设置"区域：
- 输入场景名称（如"积极达峰场景"）
- 选择模型类型：LEAP或Kaya
- 设置参数：
  - 生产总值增长率：如5%输入0.05
  - 能源效率改善率：如3%输入0.03
  - 可再生能源占比提升率：如1.5%输入0.015
- 点击"添加场景"按钮

重复以上步骤可添加多个场景（如积极、基准、保守三种）

### 4. 运行预测
- 设置预测年限（默认30年）
- 点击"运行预测"按钮

### 5. 查看结果
系统自动生成三张图表：

#### 📈 图表1：碳排放预测曲线
- 折线图显示所有场景的碳排放趋势
- 每个场景用不同颜色区分
- 峰值点用大圆点标注
- 图表下方显示每个场景的达峰年份和峰值

#### 📊 图表2：生产总值预测
- 折线图显示各场景的GDP增长曲线
- 单位自动转换为亿元

#### 🔋 图表3：能源消费预测（含能源结构）
- 堆叠柱状图
- 绿色部分：可再生能源（零排放）
- 灰色部分：非可再生能源
- 注：默认显示第一个场景的能源结构

## 🔬 模型说明

### LEAP模型
```
能源消费 = GDP × 能源强度
能源强度 = 基准能源强度 × (1 - 能源效率改善率)^t
碳排放 = 非可再生能源 × 碳排放系数
非可再生能源 = 能源消费 × (1 - 可再生能源占比)
```

**特点**：
- 侧重能源结构转换
- 明确区分可再生和非可再生能源
- 可再生能源零排放

### Kaya模型
```
CO2 = GDP × (Energy/GDP) × (CO2/Energy)
```
其中：
- GDP：生产总值
- Energy/GDP：能源强度（随效率改善下降）
- CO2/Energy：碳排放系数（随可再生能源占比增加而降低）

**特点**：
- 基于Kaya恒等式
- 分解碳排放驱动因素
- 突出能源效率和碳强度的作用

## ❓ 常见问题

### Q1: 为什么有些场景立即达峰？
A: 当能源效率改善和可再生能源发展速度大于GDP增长带来的能源需求增长时，碳排放会立即开始下降。这表示该场景下从基准年就开始减排。

### Q2: LEAP和Kaya结果相同吗？
A: 不完全相同。LEAP侧重能源结构变化，Kaya基于恒等式分解。在相同参数下，两模型结果接近但可能略有差异。

### Q3: 可再生能源占比如何影响结果？
A: 可再生能源零排放，占比越高，碳排放越低。每年提升1.5%意味着30年后可再生能源占比达到45%。

### Q4: 能否预测更长时间？
A: 可以。修改预测年限，但注意：时间越长，不确定性越大，参数假设的适用性越需要谨慎评估。

### Q5: 如何导出结果数据？
A: 目前系统主要提供可视化。如需导出，可在浏览器中右键保存图表，或查看开发者工具的Network标签获取API返回的原始数据。
