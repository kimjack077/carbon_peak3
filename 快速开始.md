# 碳达峰预测系统 - 快速开始指南

## 🚀 5分钟快速上手

### 步骤1：安装依赖

```bash
cd backend
pip install -r requirements.txt
```

### 步骤2：测试模型

```bash
# 运行测试脚本，验证LEAP和Kaya模型
python test_models.py
```

预期输出：
```
============================================================

============================================================

测试 LEAP 模型
- 预测年数: 26
- 达峰年份: 2030
- 峰值碳排放: 12500.50 万吨CO₂
...
```

### 步骤3：启动后端服务

```bash
cd backend
python app.py
```

后端将在 `http://127.0.0.1:5000` 启动。

### 步骤4：启动前端（可选）

```bash
cd frontend
npm install
npm run serve
```

前端将在 `http://localhost:8080` 启动。

### 方式2：直接使用模型（高级）

```python
from backend.model.leap_model import forecast_leap
from backend.model.kaya_model import forecast_kaya
import pandas as pd

# 准备数据
base = {
    "years": [2022, 2023, 2024],
    "gdp": [114.37, 126.06, 134.52],
    "co2": [11200.5, 11450.8, 11580.3],
    "mix": {
        "coal": [0.56, 0.54, 0.52],
        "oil": [0.18, 0.18, 0.18],
        "gas": [0.09, 0.10, 0.11],
        "ren": [0.17, 0.18, 0.19]
    },
    "power_share": [0.27, 0.29, 0.31]
}

scenario = {
    "gdp_growth_rate": 0.05,
    "efficiency_improvement_rate": 0.03,
    "coal_reduction_rate": 0.01,
    "power_share_increase": 0.02,
    "ren_share_increase": 0.015
}

# LEAP预测
results_leap, peak_leap = forecast_leap(base, scenario, 2050)
df_leap = pd.DataFrame(results_leap)

# Kaya预测
results_kaya, peak_kaya = forecast_kaya(base, scenario, 2050)
df_kaya = pd.DataFrame(results_kaya)

print(f"LEAP达峰年: {peak_leap}")
print(f"Kaya达峰年: {peak_kaya}")
```

## 📈 参数调优建议

### 保守参数（达峰较晚）
- `efficiency_improvement_rate`: 0.015 - 0.020
- `coal_reduction_rate`: 0.003 - 0.008
- `ren_share_increase`: 0.008 - 0.012

### 基准参数（标准预测）
- `efficiency_improvement_rate`: 0.020 - 0.030
- `coal_reduction_rate`: 0.008 - 0.015
- `ren_share_increase`: 0.012 - 0.018

### 积极参数（达峰较早）
- `efficiency_improvement_rate`: 0.030 - 0.050
- `coal_reduction_rate`: 0.015 - 0.030
- `ren_share_increase`: 0.018 - 0.030

## ⚠️ 常见错误

### 错误1：能源结构份额不为1

```
ValueError: 第1年能源结构份额和不为1: 0.98
```

**解决**：确保 `mix_coal + mix_oil + mix_gas + mix_ren = 1.0`

### 错误2：数据年份不连续

```
ValueError: 至少需要3年的历史数据
```

**解决**：提供连续3年的数据（如2022-2024）

### 错误3：电力占比超范围

```
ValueError: 第2年电力占比超出范围[0,1]: 1.2
```

**解决**：确保 `power_share` 在 [0, 1] 范围内
