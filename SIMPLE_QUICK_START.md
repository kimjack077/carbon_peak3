# 简化版碳达峰预测系统 - 快速开始

## 🚀 一键启动

双击运行：`start_simple.bat`

或者手动启动：

```bash
# 终端1 - 启动后端
python simple_app.py

# 终端2 - 启动前端
cd frontend
npm run serve
```

## 📊 数据格式

CSV文件包含5列（已提供示例数据）：

| 列名 | 说明 | 单位 |
|------|------|------|
| year | 年份 | - |
| energy_consumption | 能源消费 | 万吨标煤 |
| gdp | 生产总值 | 万元 |
| co2_emission | 碳排放 | 万吨CO₂ |
| renewable_ratio | 可再生能源占比 | 0-1 |

## 🎯 三步使用

### 1️⃣ 添加场景
- 场景名称：如"积极达峰"
- 模型：LEAP 或 Kaya
- 参数：
  - GDP增长率 (如 5% = 0.05)
  - 能源效率改善率 (如 3% = 0.03)
  - 可再生能源占比提升率 (如 1.5% = 0.015)

### 2️⃣ 运行预测
- 设置预测年限（默认30年）
- 点击"运行预测"按钮

### 3️⃣ 查看结果
三张图表自动生成：
- **碳排放曲线**：显示达峰年份和峰值
- **GDP预测**：经济增长趋势
- **能源结构**：可再生vs非可再生能源

## ❓ 常见问题

**Q: 为什么需要至少添加一个场景？**  
A: 系统需要至少一个场景参数来运行预测模型。

**Q: 可以同时对比多个场景吗？**  
A: 可以！添加多个场景后运行预测，所有场景的结果会在同一图表中显示。

**Q: 如何使达峰更早？**  
A: 提高能源效率改善率和可再生能源占比，或降低GDP增长率。
