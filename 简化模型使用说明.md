# 简化碳达峰预测模型使用说明

## 模型简介

这是一个专门为**只有2年历史数据**设计的简化碳达峰预测模型。适用于数据有限的情况。

### 数据要求

只需要以下基础数据：
- **年份**：如 [2020, 2021]
- **总产值（万元）**：如 [14167844.68, 17243937.42]
- **综合能耗（万吨标煤）**：如 [1120.326643, 1192.619158]
- **碳排放（万吨CO2）**：如 [5011.01, 5492.79]

### 核心算法

模型基于三个关键指标进行预测：

1. **能源强度 (EI)**：能源消费 / GDP
2. **排放因子 (EF)**：碳排放 / 能源消费（反映能源结构清洁程度）
3. **碳强度 (CI)**：碳排放 / GDP

**预测公式：**
```
碳排放 = GDP × 能源强度 × 排放因子
```

## 快速开始

### 1. 基本使用

```python
from model.simple_model import SimpleCarbonPeakModel, run_simple_analysis

# 准备数据
base_data = {
    'year': [2020, 2021],
    'gdp': [14167844.68, 17243937.42],  # 万元
    'energy': [1120.326643, 1192.619158],  # 万吨标煤
    'co2': [5011.01, 5492.79]  # 万吨CO2
}

# 运行标准情景分析
model, results, report = run_simple_analysis(base_data, horizon_year=2060)
print(report)
```

### 2. 自定义情景

```python
# 创建模型实例
model = SimpleCarbonPeakModel(base_data)

# 定义自定义情景
scenario = {
    'gdp_growth_rate': 0.05,  # GDP年增长率 5%
    'energy_intensity_reduction': 0.05,  # 能源强度年下降率 5%
    'emission_factor_reduction': 0.04  # 排放因子年下降率 4%
}

# 预测
df, peak_year = model.predict(scenario, horizon_year=2060)

if peak_year:
    print(f"达峰年份: {peak_year}年")
else:
    print("未达峰或持续下降")
```

## 情景参数说明

### 标准情景

模型提供三种预设情景：

| 情景 | GDP增长率 | 能源强度下降率 | 排放因子下降率 | 说明 |
|------|-----------|----------------|----------------|------|
| **基准情景** | 6% | 2% | 1% | 保持中等经济增长和基本减排措施 |
| **适度减排** | 5.4% | 3% | 2% | 加大节能减排力度 |
| **强化减排** | 4.8% | 4% | 3% | 大幅提升能效和清洁能源 |

### 参数调整建议

**实现碳达峰的关键条件：**

#### 1. 能源强度下降率 (4-6%)
- **技术创新**：提高能源利用效率
- **产业升级**：减少高耗能产业比重
- **节能改造**：工业、建筑、交通领域节能

#### 2. 排放因子下降率 (3-5%)
- **能源结构优化**：增加风能、太阳能、核能
- **煤炭替代**：减少煤炭使用，增加天然气和可再生能源
- **碳捕集技术**：CCS技术应用

#### 3. GDP增长率 (3-5%)
- **质量导向**：从高速增长转向高质量发展
- **合理增速**：避免盲目追求经济总量

## 重要说明

### 数据假设
- 历史GDP增长率21.71%被视为特殊情况（如疫情后恢复）
- 模型自动将长期GDP增长率上限设为6%
- 建议根据实际情况调整参数

### 模型局限性
- 仅基于2年数据，趋势外推的不确定性较大
- 未考虑政策突变、技术突破等外部因素
- 假设各参数线性变化，实际可能非线性

### 最佳实践
1. **多情景分析**：运行多个情景，比较不同政策组合
2. **敏感性分析**：测试关键参数变化对结果的影响
3. **定期更新**：有新数据时及时更新模型
4. **结合实际**：预测结果需结合本地实际情况解读
