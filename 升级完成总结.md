## 🚀 核心功能

### 1. LEAP模型
```python
碳排放 = 经济规模(GDP) × 能源强度(EI) × 排放因子(CF)
```
- **适用**：详细能源结构分析
- **特点**：自下而上，技术路径清晰

### 2. Kaya模型
```python
CO2 = 人口 × (GDP/人口) × (能源/GDP) × (CO2/能源)
```
- **适用**：宏观政策效果评估
- **特点**：因子分解，驱动力分析

### 3. 新增参数

| 参数 | 说明 | 典型值 |
|------|------|--------|
| model_type | 模型类型 | "leap" / "kaya" |
| gdp_growth_rate | GDP增长率 | 0.05 (5%/年) |
| population_growth_rate | 人口增长率 | 0.00 |
| efficiency_improvement_rate | 能效改善率 | 0.03 (3%/年) |
| coal_reduction_rate | 煤炭下降率 | 0.01 (1%p/年) |
| power_share_increase | 电力占比提升 | 0.02 (2%p/年) |
| ren_share_increase | 可再生占比提升 | 0.015 (1.5%p/年) |
| grid_cf_decline | 电网碳因子下降 | 0.05 (5%/年) |

## 📈 使用示例

### Python直接调用
```python
from backend.model.leap_model import forecast_leap

base = {
    "years": [2022, 2023, 2024],
    "gdp": [114.37, 126.06, 134.52],
    "co2": [11200.5, 11450.8, 11580.3],
    "mix": {
        "coal": [0.56, 0.54, 0.52],
        "oil": [0.18, 0.18, 0.18],
        "gas": [0.09, 0.10, 0.11],
        "ren": [0.17, 0.18, 0.19]
    },
    "power_share": [0.27, 0.29, 0.31]
}

scenario = {
    "gdp_growth_rate": 0.05,
    "efficiency_improvement_rate": 0.03,
    "coal_reduction_rate": 0.01,
    "power_share_increase": 0.02,
    "ren_share_increase": 0.015
}

results, peak_year = forecast_leap(base, scenario, 2050)
print(f"达峰年: {peak_year}")
```

## 🔧 故障排查

### 问题1：模型测试失败
```bash
ModuleNotFoundError: No module named 'pandas'
```
**解决**：`pip install -r backend/requirements.txt`

### 问题2：数据格式错误
```
ValueError: 能源结构份额和不为1
```
**解决**：检查 `mix_coal + mix_oil + mix_gas + mix_ren = 1.0`

### 问题3：达峰年异常
```
达峰年: None（未达峰）
```
**解决**：
- 增大 `efficiency_improvement_rate`
- 增大 `coal_reduction_rate`  
- 增大 `ren_share_increase`
